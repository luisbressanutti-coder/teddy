# Instrucciones para generar plantillas SVG A4 desde Blender

Estas instrucciones describen cómo usar Blender para exportar modelos de papel de un oso de peluche con un tamaño de 50cm. 

## Pasos
1. **Abre Blender:** Lanza la aplicación.
2. **Carga el modelo:** Importa o crea el modelo 3D que deseas exportar.
3. **Configura la exportación:**
   - Ve a "Configuración de exportación" en Blender.
   - Selecciona "Modelo de papel" como el tipo de exportación.
4. **Ajusta la configuración:**
   - **Cartón de 2mm:** Asegúrate de que el grosor esté configurado para 2mm.
   - **Pestañas internas:** Habilita las pestañas internas para facilitar el ensamblaje.
   - **Líneas de plegado marcadas:** Activa esta opción para que las líneas de plegado sean visibles en el modelo.
   - **Escala de 500mm:** Ajusta la altura del modelo a 500mm para asegurarte de que sea del tamaño correcto.
5. **Exporta:** Haz clic en "Exportar" para generar tus plantillas.
6. **Revisa tus archivos SVG:** Asegúrate de que se hayan generado correctamente en páginas A4 individualmente.

Ahora puedes imprimir tus plantillas y empezar a crear tu oso de peluche.
