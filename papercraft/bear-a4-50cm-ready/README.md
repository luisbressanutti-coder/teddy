# Low-Poly Teddy Bear Assembly Instructions

This document contains instructions for assembling the low-poly teddy bear templates provided in this folder. Follow the steps below carefully to achieve a sturdy and beautiful result.

## Materials Needed:
- 2mm thick cardboard
- Craft knife or scissors
- Cutting mat
- Glue or double-sided tape
- Bone folder (optional)
- Ruler
- Pencil

## Instructions:
1. **Download and Print**: Use the SVG files provided to print each template on 2mm cardboard.

2. **Cut Out Templates**: Using a craft knife or scissors, carefully cut along the lines of each template.
   - Make sure to cut out all tabs and edges properly.

3. **Folding**: Once cut, fold all edges and tabs where indicated. You can use a bone folder for crisp folds.

4. **Assemble**: Start with the body parts and gradually add each piece together, securing with glue or tape as needed.

5. **Final Touches**: After assembly, make sure all edges are securely glued and that the bear is sturdy.

Enjoy your crafting!
